# Epic-4: PDF Generation Service
**Description:** Backend service to generate non-repudiable legal PDFs for undertakings, receipts, and exit certificates.
**Status:** DRAFT
**Owner:** Developer
**Links:** [Architecture Section 2]

## User Stories
1. **Generate Undertaking:** As a System, I want to generate a signed PDF of the student undertaking so that we have a legal record.
2. **Generate Fee Receipt:** As a Student, I want to download a PDF receipt for my payment so that I have proof of transaction.
3. **Store in S3:** As a System, I want to store generated PDFs in a private S3 bucket so that they are secure and auditable.
