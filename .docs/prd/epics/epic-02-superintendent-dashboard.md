# Epic-2: Superintendent Dashboard & Approval
**Description:** The internal dashboard for Superintendents to review applications, schedule interviews, and approve/reject candidates.
**Status:** DRAFT
**Owner:** PM
**Links:** [PRD Section 4.F]

## User Stories
1. **View Applications:** As a Superintendent, I want to see a filtered list of new/pending applications so that I can process them.
2. **Review Candidate:** As a Superintendent, I want to view a candidate's details and documents so that I can verify eligibility.
3. **Schedule Interview:** As a Superintendent, I want to schedule a physical or online interview so that Trustees can evaluate the candidate.
4. **Approve/Reject:** As a Superintendent, I want to update the application status so that the candidate is notified.
