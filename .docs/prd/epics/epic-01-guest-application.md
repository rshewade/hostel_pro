# Epic-1: Guest-First Application Tracking
**Description:** Implement the public-facing application system where applicants can submit forms and track status via OTP without creating a permanent user account.
**Status:** DRAFT
**Owner:** PM
**Links:** [PRD Section 4.B]

## User Stories
1. **Apply for Admission:** As an applicant, I want to submit an admission form with my mobile number so that I can start the process.
2. **Verify Mobile OTP:** As an applicant, I want to verify my mobile via OTP so that my submission is authenticated.
3. **Track Status:** As an applicant, I want to check my status using my Tracking ID and OTP so that I know if I'm selected.
4. **Upload Documents:** As an applicant, I want to upload required PDF/JPG documents so that my application is complete.
