# Epic-3: Role-Based Access Control (RBAC) & Auth
**Description:** Implement the secure authentication system using Supabase Auth, including distinct roles for Students, Parents, Staff, and Trustees.
**Status:** DRAFT
**Owner:** Architect
**Links:** [Architecture Section 6.1]

## User Stories
1. **Staff Login:** As a Staff member, I want to log in with email/password so that I can access the admin panel.
2. **Student Activation:** As an approved student, I want to set my password for the first time so that I can access my dashboard.
3. **Parent OTP Access:** As a parent, I want to log in via Mobile OTP so that I can view my ward's leave status.
4. **Role Guards:** As a Developer, I want to ensure API endpoints are protected by Role guards so that data is secure.
